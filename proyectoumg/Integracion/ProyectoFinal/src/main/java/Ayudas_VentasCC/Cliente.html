Mantenimiento Clientes

Este modulo nos provee el mantenimiento de los clientes, donde usted podra ingresar, eliminar, modificar, actualizar e imprimir la lista de clientes, todo esto se maneja en base al id de cliente que se le asigne, como sabemos un mantenimiento llevara TODA informacion clave es por ello que debe llenar todos los campos que se le presentan, la interfaz presentada es la siguiente :




Ahora desglosemoslo por partes para entender su funcionamiento:
1) Llenado de campos

Como primer punto, usted debe llenar cada uno de los campos que se le presentan, para poder llevar un esctricto control de la base de datos y sistema:




2) Informacion asignada

Luego de llenar los campos, en el panel derecho se vera la asignacion de informacion que introdujo y se ordenan a manera de tabla.¡EL ID DEL CLIENTE SE ASIGNARA DE FORMA AUTOMATICA!


3) Manejo del CRUD:

En esta seccion, usted puede darle su respectivo mantenimiento a los clientes, ya sea que desee insertar uno nuevo, editar, eliminar, imprimir o consultar uno existente, para poder realizarlo no olvide de colocar el ID DEL CLIENTE ya que este es como el codigo unico que identifica a quien se dirige la modificacion



4) Adicionales

En esta seccion encontrara la ayuda y los reportes, este ultimo contiene el resumen descargable de todas los registros de clientes que ha hecho hasta el momento de forma ordenada, para su facil comprension y forma de compartir.
