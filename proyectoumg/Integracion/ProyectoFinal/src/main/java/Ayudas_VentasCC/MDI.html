MDI CuentasXc

En el MDI encontramos las opciones principales con las que cuenta el sistema. En este MDI contamos con 2 apartados principales que son "Catalogos y Procesos" cada uno almacena otras opciones para navegar y permitirá realizar una serie de actividades las cuales tienen como objetivo hacer que el sistema sea fácil de utilizar. Tambien cuenta con dos botones que facilitan el manejo.




Catalogos

En el apartado de Catalogos tenemos 3 opciones diferentes que podemos escoger, al dar click sobre cualquier opcion realizara un llamado al manetnimiento clickeado y nos trasladara a su Mantenimiento.

Las diferentes Opciones son:
Clientes
Vendedores
Ventas




Procesos
