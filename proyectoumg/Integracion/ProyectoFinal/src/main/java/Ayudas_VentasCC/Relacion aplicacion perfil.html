Relacion Aplicacion perfil

Este transaccional esta dedicado para que el usuario asigne diferentes aplicaciones a los usuarios designados, usted podra colocar cuantas aplicaciones desee a un perfil en especifico, a continuacion la interfaz que usted ve es la siguiente:




Ahora desglosemoslo por partes para entender su funcionamiento:
1) Asignacion de perfil

Como primer punto, usted podra elegir el perfil del usuario que desee, por medio de su codigo de identificacion, al apar se le mostrara cual es el cargo que ejerce ese codigo de perfil que ha seleccionado, para que de esta manera usted pueda elegir las aplicaciones deseadas al empleado:




2) Asignacion de aplicaciones

Luego de seleccionar el perfil, debemos elegir que aplicaciones o cargos le daremos a la persona seleccionada, para ello debe posicionarse en el lado derecho sobre el contenedor de aplicaciones y seleccionar la que desea, o bien puede seleccionar todas.

Para asignar aplicaciones:

  • Para seleccionar solo una aplicacion se utiliza:▶
  • Para seleccionar TODAS las aplicaciones se utiliza:▶▶

Del mismo modo puede eliminar o desasignar UNA o TODAS las aplicaciones:
  • Para eliminar solo una aplicacion se utiliza:◀
  • Para eliminar TODAS las aplicaciones se utiliza:◀◀

NOTAS ADICIONALES:

El perfil ya tiene aplicaciones asignadas:

Si usted al momento de seleccionar el perfil, automaticamente le aparecen items en el panel de aplicaciones asignadas es porque este perfil ya tiene asignaciones anteriormente, usted puede modificarlo de igual forma utilizando los botones de eliminar y asignar



3) Asignacion de permisos

Luego que ha seleccionado el perfil junto a su respectiva aplicacion debera habilitar el panel de permisos, este consiste en que el perfil encargado estara limitado a realizar ciertas acciones del CRUD, luego de asegurarse de seleccionar los permisos necesarios, presione en confirmar, y ¡automaticamente se le asignara la fecha en que realizo la accion!



4) Impresion de reportes

En esta seccion usted vera en formato para impresion las asignaciones que se han hecho, usted puede imprimirlas o realizar otras acciones que crea convenientes
